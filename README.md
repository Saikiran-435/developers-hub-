# Developers Hub 

Developers Hub is a MERN stack freelancer platform that enables developers to create and manage profiles, explore other developers based on skills, and collaborate on freelance work.
The platform allows users to review developer profiles, assign tasks according to expertise, and provide ratings and feedback after task completion based on overall satisfaction.

---

##  Features

- User registration & login (JWT authentication)
- View all developer profiles
- View individual developer profile
- Rate other developers 
- Average rating calculation
- Secure APIs with middleware
- MongoDB database integration

---

##  Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Token (JWT)
